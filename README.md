# HardwareMastery V2

A Clash Royale clone for tracking electronics/PCB design study - gamified learning experience.

## Features

- Arena-centric home screen with CSS art environments
- 12 arenas with progression system
- 13 character cards (fantasy CSS/SVG warriors)
- 6 boss characters
- Combat-style quiz battles via Gemini Flash API
- Trophy progression system
- PWA support
- Real-time sync with Firebase

## Tech Stack

- Vite + React 19
- Tailwind CSS 4
- Framer Motion
- Lucide React
- Firebase Firestore
- Zustand
- Gemini Flash API

## Setup

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your Firebase config and Gemini API key
4. Run `npm run dev`
